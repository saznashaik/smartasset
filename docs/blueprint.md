# **App Name**: AssetAI Central

## Core Features:

- Dashboard Integration: Integrate QuickSuite dashboards for a comprehensive view of asset metrics.
- AI Chat Integration: Embed Agent AI chat functionality within the app for intelligent assistance.
- Executive Summary: Provide an overview of total assets, high-risk items, and repair costs at a glance.
- Predictive Analysis Tool: Use AI to predict potential asset failures based on historical data and real-time metrics.

## Style Guidelines:

- Primary color: Deep ocean blue (#1E3A8A) to convey stability and trust.
- Background color: Light gray (#F5F5F5), almost white, for a clean, modern look.
- Accent color: Vivid cerulean blue (#1CB0F6) to draw attention to important actions and data visualizations.
- Headline font: 'Poppins', a geometric sans-serif, for a precise and contemporary feel; use 'Inter' for body text.
- Use crisp, line-based icons for asset types, risk indicators, and actions.
- Employ a clear, card-based layout for dashboard metrics, ensuring easy readability and navigation.
- Subtle transitions and animations for loading data and displaying updated information.
